# Churn_rate
This Repository consist of implementation of a deep learning model which classify the customers of bank. The model predicts if a customer will close his/her's bank account given various parameters in csv sheet


The csv file cosist of the various parameters of a customer like credit score, Account Balance, Geographical location, age, gender, tenure etc and the data wether he closed his account or not.

The ipynb file cosist of a deep learning model based on Artificial Neural Network which predicts the churn rate given a data. It also consist of other models for comparision like decision tree classifier and Random Forest Classifier
